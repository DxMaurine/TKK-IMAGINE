📋 Pendahuluan

Selamat datang di TKK IMAGINE. Kami menghargai privasi Anda dan berkomitmen untuk melindungi informasi pribadi yang Anda berikan kepada kami. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan layanan AI Image Generator kami.

📊 Informasi yang Kami Kumpulkan

Data yang Disimpan Secara Lokal:

  • Prompt Teks: Deskripsi gambar yang Anda masukkan untuk menghasilkan AI image
  • Riwayat Gambar: Gambar yang telah Anda buat (disimpan di browser Anda)
  • Preferensi Pengguna: Pengaturan aplikasi dan preferensi tampilan
  • Data Favorit: Prompt dan gambar yang Anda tandai sebagai favorit

Data yang Tidak Kami Kumpulkan:

  • Informasi pribadi seperti nama, email, atau nomor telepon
  • Data lokasi atau informasi perangkat yang sensitif
  • Riwayat browsing di luar aplikasi kami

💾 Cara Kami Menyimpan Data

Local Storage: Semua data Anda disimpan secara lokal di browser Anda menggunakan teknologi localStorage. Ini berarti:

  • Data tidak dikirim ke server kami
  • Data hanya dapat diakses dari browser yang sama
  • Anda memiliki kontrol penuh untuk menghapus data kapan saja
  • Data akan hilang jika Anda menghapus cache browser

🔗 Layanan Pihak Ketiga

Pollinations.ai API: Kami menggunakan layanan Pollinations.ai untuk menghasilkan gambar AI. Ketika Anda membuat gambar:

  • Prompt teks Anda dikirim ke Pollinations.ai untuk diproses
  • Pollinations.ai memiliki kebijakan privasi tersendiri
  • Kami tidak menyimpan atau mengontrol data di server Pollinations.ai
  • Silakan baca kebijakan privasi Pollinations.ai untuk informasi lebih lanjut

🎯 Penggunaan Data

Data yang kami kumpulkan digunakan untuk:

  • Fungsionalitas Aplikasi: Menyimpan riwayat dan preferensi Anda
  • Pengalaman Pengguna: Memberikan fitur auto-save dan manajemen prompt
  • Optimasi Layanan: Meningkatkan performa dan fitur aplikasi

🛡️ Keamanan Data

Kami mengimplementasikan langkah-langkah keamanan berikut:

  • HTTPS: Semua komunikasi dienkripsi
  • Local Storage: Data disimpan secara lokal, tidak di server
  • No Registration: Tidak memerlukan pendaftaran atau login
  • Open Source: Kode dapat diaudit untuk transparansi

👤 Hak Pengguna

Sebagai pengguna, Anda memiliki hak untuk:

  • Mengakses Data: Melihat semua data yang tersimpan di browser Anda
  • Menghapus Data: Menghapus sebagian atau seluruh data kapan saja
  • Mengekspor Data: Mengunduh data Anda dalam format JSON
  • Kontrol Penuh: Mengelola penyimpanan data sesuai keinginan

🔄 Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan:
